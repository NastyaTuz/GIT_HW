1.  Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно? 

Ответ - предварительно должен быть сгенерирован и подключен к GitHub ключ, создан соответствующий репозиторий. Для копирования репозитория на свой компьютер
необходимо  скопировать ссылку ssh GitHub реп. И командой clone вставить ссылку в терминале. Проверить создался ли он в нужной директории компьютера.

2.Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

Ответ - в терминале перейти в нужный каталог и с помощью команды ‘git init’ инициализировать новый реп.. Перейти в GitHub и создать новый реп. Скопировать ssh ссылку и с помощью команды ‘git remote add origin’ добавить скопированную ссылку реп. 
Далее проверить статус и ввести ‘git add .’. Для Изменений ввести ‘git commit -m ‘’ ’. Отправить кормит с помощью команды ‘git push’

3.Как получить на локальный компьютер изменения с github.com
Ответ - Перейти в свой каталог и ввести команду ‘git pull’

4. Какую информацию нам дают команда :
 - git status 
Ответ - отслеживает изменение файлов 

- git log 
Ответ - отслеживает историю изменений коммитов

- git remote -v 
Ответ - позволяет просматривать список удаленных репозиториев, ключ -v дат возможность посмотреть адреса для чтения и записи, которые привязаны к реп.
